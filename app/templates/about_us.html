Know Your Heart

A.I. based Prediction of Heart Disease

User supplied input and recieved response

Overview

  • This project is a web application that uses machine learning to analyze the heart disease dataset and make predictions based on supplied medical data.
  • The website should display the risk factors to the user as well as a probability or confidence score of their health in the coming years. If possible, the lifestyle choices to be made should also be displayed.

Main Modules

  1. Machine Learning

    2. Tools and Technologies:

    • Libraries - NumPy, pandas, Scikit-Learn, Matplotlib, seaborn

  2. Back-end and Integration

    3. Tools and Technologies:

    • Backend - Flask tool is used which is a micro-web framework - libraries used are render_template, Requests, URL_for, Redirect
    • Database - Pymongo is used which is a server-based database
    • Integration - Docker
    • Deployment - Amazon and Google cloud platforms (AWS and GCP)

  3. Front-end

    4. Tools and Technologies:

    • Front-end - HTML, CSS, JavaScript
    • Testing - Selenium, Pytest
